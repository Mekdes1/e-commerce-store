# E-commerce Store
This is a small women’s clothing and shoe shopping application.

## Description
Each available product is rendered in the home page of the application.
It gives the user the ability to search for available products based on their specific name.
It gives the user the ability to add and delete items in their cart, click on the shopping basket to see the changes you have made(add/delete).
It is possible to add the same item multiple times to the cart.
